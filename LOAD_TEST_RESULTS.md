## 주요 발견사항

1. MySQL INSERT 경합

   - 인기도 높은 콘서트 동시 예약에서 병목
   - 권장 해결: 복합 인덱스 추가, 파티션 테이블 도입
   - 예상 효과: 응답시간 70% 개선

2. Kafka Producer 큐 오버플로우

   - High Load에서 메시지 발행 지연 (평균 50ms)
   - 권장 해결: 파티션 3 -> 6으로 증가
   - 예상 효과: 발행 지연 70% 감소

3. 메모리 선형 증가

   - VU 100: 350MB, VU 300: 950MB, VU 1000: 1,800MB
   - 권장 해결: GC 튜닝, 커넥션 풀 조정
   - 예상 효과: 메모리 효율 30% 향상

4. CPU 포화 시 성능 급감
   - CPU 70% 이상에서 응답시간 급증
   - 권장 해결: 비동기 처리 강화, 스레드 풀 최적화
   - 예상 효과: 처리량 2-3배 향상
